# ML-DL-codes-and-models-for-Hackathon

 Project Overview

This repository contains Machine Learning (ML) and Deep Learning (DL) based cybersecurity fraud detection modules, developed for hackathon and academic purposes.

The project focuses on detecting:

1) Phishing URLs

2) Fraudulent / suspicious emails

3) Insider threat risks

All individual risk signals are combined to generate a final risk decision.
